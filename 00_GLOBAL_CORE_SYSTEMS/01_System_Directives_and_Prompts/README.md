# System Directives and Prompts

The live system directives for this workspace stay where the tools read them:

- **Standing instructions:** `/CLAUDE.md` (repo root). Claude Code reads it from the root; do not move it.
- **BUBU skill prompts and hooks:** `/bubu-toolkit/` (repo root). Skills auto-discover from `bubu-toolkit/skills/`; the installable bundle is `/bubu-toolkit.plugin`.
- **Writing voice:** `bubu-toolkit/skills/bubu-stopslop/SKILL.md` (the StopSlop editorial spec).

This folder holds any future system prompts that are not tied to a runtime location. A stale duplicate copy of bubu-toolkit that briefly lived here was removed on 2026-07-07; the root copy is canonical.
